# 주문 조회

> 원문: https://apicenter.commerce.naver.com/docs/commerce-api/current/주문-조회

주문 조회 API

[## 📄️ 변경 상품 주문 정보 구조체

이 구조체는 주문건의 변경 상품 주문 정보를 표현하는 구조체입니다.](/docs/commerce-api/current/schemas/변경-상품-주문-정보-구조체)

[## 📄️ 상품 주문 정보 구조체

이 구조체는 주문건의 상세 정보를 표현하는 구조체입니다.](/docs/commerce-api/current/schemas/상품-주문-정보-구조체)

[## 📄️ 상품 주문 목록 조회

주문에 대한 상품 주문 번호 목록을 조회합니다.](/docs/commerce-api/current/seller-get-product-order-ids-pay-order-seller)

[## 📄️ 조건형 상품 주문 상세 내역 조회

조건에 맞는 상품 주문에 대한 상세 내역을 조회합니다.](/docs/commerce-api/current/seller-get-product-orders-with-conditions-pay-order-seller)

[## 📄️ 변경 상품 주문 내역 조회

조회 요청 범위의 기준은 변경 일시(date-time)입니다.](/docs/commerce-api/current/seller-get-last-changed-status-pay-order-seller)

[## 📄️ 상품 주문 상세 내역 조회

상품 주문에 대한 상세 상품 주문 내역을 조회합니다. 요청 가능한 상품 주문 번호는 최대 300개입니다.](/docs/commerce-api/current/seller-get-product-orders-pay-order-seller)
