[## 📄️ 주문-클레임 처리 반환 구조체

이 구조체는 주문-클레임 처리 관련 다수 API에서 공통으로 사용하는 반환 메시지 형식을 표현하는 구조체입니다.](/docs/commerce-api/current/schemas/주문-클레임-처리-반환-구조체)

[## 📄️ 발주 확인 처리

단수 또는 복수 개 상품 주문의 발주를 확인 처리합니다. 요청 가능한 상품 주문 번호는 최대 30개입니다.](/docs/commerce-api/current/seller-confirm-placed-product-orders-pay-order-seller)

[## 📄️ 발송 처리

단수 또는 복수 개 상품 주문을 발송 처리합니다. 요청 가능한 상품 주문 번호는 최대 30개입니다.](/docs/commerce-api/current/seller-dispatch-product-orders-pay-order-seller)
