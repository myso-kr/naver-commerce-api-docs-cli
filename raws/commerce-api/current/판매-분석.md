# 판매 분석

> 원문: https://apicenter.commerce.naver.com/docs/commerce-api/current/판매-분석

[브랜드스토어 전용] 커머스솔루션마켓에서 API데이터솔루션 구독 후 이용가능합니다. 솔루션을 추가하지 않고 API 호출 시 응답결과가 제한됩니다.
최대 18개월 전 데이터부터 제공됩니다.

[## 📄️ 배송 통계 API

[브랜드스토어 전용] 배송 유형별 주문량과 배송 현황 데이터를 제공합니다.](/docs/commerce-api/current/delivery-report-using-get-bizdata-stats)

[## 📄️ 판매 성과 API

[브랜드스토어 전용] 시간대별 스토어의 주요 거래 데이터를 제공합니다.](/docs/commerce-api/current/sales-report-using-get-bizdata-stats)

[## 📄️ 상품/마케팅 채널 카테고리(소)별 API

[브랜드스토어 전용] 카테고리(소)별로 최대 50개의 마케팅 채널별 결제 데이터를 결제액 기준 상위 15개 카테고리에 대해 제공합니다.](/docs/commerce-api/current/product-category-channel-report-using-get-bizdata-stats)

[## 📄️ 상품/마케팅 채널 API

[브랜드스토어 전용] 마케팅 채널별 상품 결제 데이터를 결제액 기준 상위 1000개 제공합니다.](/docs/commerce-api/current/product-marketing-report-using-get-bizdata-stats)

[## 📄️ 상품/검색 채널 API

[브랜드스토어 전용] 검색 채널별 상품 결제 관련 데이터를 결제액 기준 상위 1000개 제공합니다.](/docs/commerce-api/current/product-search-report-using-get-bizdata-stats)

[## 📄️ 상품/검색 채널 상품별 키워드 API

[브랜드스토어 전용] 상품별로 판매에 영향을 주는 주요 키워드 데이터 최대 5개를 결제액 기준 상위 10개 상품에 대해 제공합니다.](/docs/commerce-api/current/keyword-by-product-report-using-get-bizdata-stats)

[## 📄️ 상품 성과 API

[브랜드스토어 전용] 상품별 결제 데이터를 결제액 기준 상위 1000개 제공합니다.](/docs/commerce-api/current/product-unit-report-using-get-bizdata-stats)
