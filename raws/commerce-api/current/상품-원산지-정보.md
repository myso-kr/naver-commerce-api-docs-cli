상품 원산지 정보 API

[## 📄️ 원산지 코드 정보 전체 조회

원산지 코드 전체를 조회합니다.](/docs/commerce-api/current/get-all-origin-area-list-product)

[## 📄️ 원산지 코드 정보 다건 조회

원산지 상세 지역명을 입력하거나 원산지 코드를 입력하여 검색할 수 있습니다. 원산지 코드를 입력한 경우 다른 검색 조건은 무시합니다.](/docs/commerce-api/current/get-origin-area-list-product)

[## 📄️ 하위 원산지 코드 정보 다건 조회

하위 원산지 코드 정보를 조회합니다. 원산지 코드를 입력하지 않으면 최상위 코드 정보를 조회합니다.](/docs/commerce-api/current/get-sub-origin-area-list-product)
