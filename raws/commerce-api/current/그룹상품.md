# 그룹상품

> 원문: https://apicenter.commerce.naver.com/docs/commerce-api/current/그룹상품

상품 API

[## 📄️ (v2) 그룹상품 등록

비동기로 동작합니다. 요청 결과는 그룹상품 요청 결과 조회 API로 확인할 수 있습니다.](/docs/commerce-api/current/create-product-1-product)

[## 📄️ (v2) 그룹상품 요청 결과 조회

그룹상품 등록 API와 그룹상품 수정 API의 요청 결과를 조회합니다. - type : CREATE(등록), UPDATE(수정)](/docs/commerce-api/current/get-progress-product)

[## 📄️ (v2) 상품 상세 정보 임시 저장

그룹상품 요청 시 개별 상품 상세 정보 저장이 필요한 경우 사전에 임시 저장합니다.](/docs/commerce-api/current/save-detail-info-product)

[## 📄️ (v2) 그룹상품 조회

그룹상품을 조회합니다.](/docs/commerce-api/current/read-product-product)

[## 📄️ (v2) 그룹상품 수정

비동기로 동작합니다. 요청 결과는 그룹상품 요청 결과 조회 API로 확인할 수 있습니다.](/docs/commerce-api/current/update-product-product)

[## 📄️ (v2) 그룹상품 삭제

그룹상품을 삭제합니다.](/docs/commerce-api/current/delete-product-product)
