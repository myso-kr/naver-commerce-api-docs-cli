# 소개

> 원문: https://apicenter.commerce.naver.com/docs/introduction

**커머스API**는 스마트스토어의 주요 기능 또는 콘텐츠를 웹 프로토콜(HTTP API)로 호출해 사용할 수 있도록 공개된 API입니다.

## 들어가기 전에[​](#들어가기-전에 "Direct link to 들어가기 전에")

커머스API를 사용하려면 커머스API센터에 가입 후 애플리케이션을 등록해 사용 권한을 획득해야 합니다. 커머스API센터 가입부터 솔루션 운영까지 자세한 내용은 [커머스API센터 안내](https://notice.naver.com/notices/cac/16928)를 참고 바랍니다.

개발사는 어드민 도구를 이용해 애플리케이션을 등록한 후 '커머스플랫폼 API 게이트웨이'를 통해 커머스API를 호출하며, API 호출 시 인증 과정을 수행해야 합니다. 자세한 내용은 본 문서의 [인증](/docs/auth) 항목에서 설명합니다.
