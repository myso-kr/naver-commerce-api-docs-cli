# 하위 원산지 코드 정보 다건 조회



```
> **GET** `https://api.commerce.naver.com/external/v1/product-origin-areas/sub-origin-areas`
```

하위 원산지 코드 정보를 조회합니다. 원산지 코드를 입력하지 않으면 최상위 코드 정보를 조회합니다.

## Request

**GET** `/v1/product-origin-areas/sub-origin-areas`
### Query Parameters

**code** string

하위 원산지 코드. 조회하고자 하는 하위 원산지의 원산지 코드를 입력합니다.
