# (v2) 제한 태그 여부 조회



```
> **GET** `https://api.commerce.naver.com/external/v2/tags/restricted-tags`
```

제한 태그 여부를 조회합니다.

## Request

**GET** `/v2/tags/restricted-tags`
### Query Parameters

**tags** string[]required

## Responses

**GET** `/v2/tags/restricted-tags` — 응답
성공

**tag**태그 (string)

**restricted**제한 태그 여부 (boolean)

제한 태그인 경우 값이 true입니다.

```json
[
  {
    "tag": "string",
    "restricted": true
  }
]
```
