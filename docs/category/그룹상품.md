# 그룹상품

상품 API

## 관련 문서

| 문서 | 설명 |
|------|------|
| [(v2) 그룹상품 등록](../api/v2/standard-group-products.POST.md) | 비동기로 동작합니다. 요청 결과는 그룹상품 요청 결과 조회 API로 확인할 수 있습니다. |
| [(v2) 그룹상품 요청 결과 조회](../api/v2/standard-group-products/status.GET.md) | 그룹상품 등록 API와 그룹상품 수정 API의 요청 결과를 조회합니다. - type : CREATE(등록), UPDATE(수정) |
| [(v2) 상품 상세 정보 임시 저장](../api/v2/standard-group-products/temp-detail-content.POST.md) | 그룹상품 요청 시 개별 상품 상세 정보 저장이 필요한 경우 사전에 임시 저장합니다. |
| [(v2) 그룹상품 조회](../api/v2/standard-group-products/{groupProductNo}.GET.md) | 그룹상품을 조회합니다. |
| [(v2) 그룹상품 수정](../api/v2/standard-group-products/{groupProductNo}.PUT.md) | 비동기로 동작합니다. 요청 결과는 그룹상품 요청 결과 조회 API로 확인할 수 있습니다. |
| [(v2) 그룹상품 삭제](../api/v2/standard-group-products/{groupProductNo}.DELETE.md) | 그룹상품을 삭제합니다. |
