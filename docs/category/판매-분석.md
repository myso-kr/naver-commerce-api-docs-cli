# 판매 분석

[브랜드스토어 전용] 커머스솔루션마켓에서 API데이터솔루션 구독 후 이용가능합니다. 솔루션을 추가하지 않고 API 호출 시 응답결과가 제한됩니다.
최대 18개월 전 데이터부터 제공됩니다.

| [판매 성과 API](../api/v1/bizdata-stats/channels/{channelNo}/sales/hourly/detail.GET.md) | [브랜드스토어 전용] 시간대별 스토어의 주요 거래 데이터를 제공합니다. |
