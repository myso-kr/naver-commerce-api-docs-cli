# 쇼핑행동 분석

[브랜드스토어 전용] 커머스솔루션마켓에서 API데이터솔루션 구독 후 이용가능합니다. 솔루션을 추가하지 않고 API 호출 시 응답결과가 제한됩니다.
최대 18개월 전 데이터부터 제공됩니다.

## 관련 문서

| 문서 | 설명 |
|------|------|
| [페이지별 API](../api/v1/bizdata-stats/channels/{channelNo}/shopping/page/detail.GET.md) | [브랜드스토어 전용] 조회 기간 내 페이지별 고객의 주요 행동 데이터를 PV 기준 상위 1000개 제공합니다. |
| [상품별 API](../api/v1/bizdata-stats/channels/{channelNo}/shopping/product/detail.GET.md) | [브랜드스토어 전용] 조회 기간 내 상품별 고객의 주요 행동 데이터를 PV 기준 상위 20개 제공합니다. |
