# 고객 데이터

[브랜드스토어 전용] 커머스솔루션마켓에서 API데이터솔루션 구독 후 이용가능합니다. 솔루션을 추가하지 않고 API호출시 응답결과가 제한됩니다. 최대 18개월 전 데이터부터 제공됩니다.

## 관련 문서

| 문서 | 설명 |
|------|------|
| [고객 현황(계정) API](../api/v1/customer-data/customer-status/account/statistics.GET.md) | [브랜드스토어 전용] 스토어의 일별 고객 현황 데이터를 제공합니다. |
| [고객 현황(채널) API](../api/v1/customer-data/customer-status/channels/{channelNo}/statistics.GET.md) | [브랜드스토어 전용] 채널의 일별 고객 현황 데이터를 제공합니다. |
| [재구매 통계 API](../api/v1/customer-data/repurchase/account/statistics.GET.md) | [브랜드스토어 전용] 스토어의 주별 재구매 데이터를 제공합니다. 매주 월요일에 지난주 데이터가 생성됩니다. API 호출 시 반드시 월요일이 포함되어 있어야 데이터가 정상적으로 호출됩니다. |
